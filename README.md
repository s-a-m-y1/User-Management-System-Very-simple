# Windows Forms Login & Registration System

A sleek, C#-based desktop application featuring a multi-form user authentication system. This project demonstrates UI/UX design in WinForms, event handling, and form navigation.

## 🚀 Features
* **Main Dashboard:** Centralized "Welcome" screen with real-time clock integration.
* **User Authentication:** Dedicated Login and Registration forms.
* **Dynamic UI:** Custom dark-themed aesthetics with background image integration.
* **Validation:** Implementation of user registration workflows and success notifications.

## 🛠️ Built With
* **Language:** C#
* **Framework:** .NET / Windows Forms (WinForms)
* **IDE:** Visual Studio
